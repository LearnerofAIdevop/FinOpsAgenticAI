Avg Maturity Indicator 10%

Process Discovery — As-is process maps created manually; pain points catalogued across O2C, P2P, R2R, FP&A
Opportunity Register — Candidate use-cases scored by effort and ROI; prioritised backlog drafted

Manual Controls — All financial controls are 100% human-executed; no automated testing in place

Risk Assessment — Initial AI risk register created; data privacy and model bias risks identified

Awareness Stage — Finance leadership aware of AI potential; executive sponsor identified
No Process Change — Day-to-day operations unchanged; agents not touching live transactions
Culture Baseline — Change management plan drafted; resistance anticipated and acknowledged
Investment Case — Business case for Phase 1 agents under development with draft ROI model

Data Audit — ERP data quality reviewed; critical gaps in master data identified (duplicates, missing fields)
Siloed Systems — Finance systems (ERP, CRM, spreadsheets) not integrated; data pulled manually
No API Layer — No middleware or integration platform in place; export/import workflows prevalent
Data Readiness Score — Initial data readiness assessment completed; cleansing roadmap drafted

Augmentation
Avg Maturity Indicator 25%

Predictive Alerts — KPI Monitoring Agent sends threshold alerts on DSO, cash balance, and budget deviation

Audit Trail Capture — Agent activity logged with timestamps and reasoning; improves evidence quality
Policy Checks — Expense policy and payment terms validations checked by AI; results presented to approver
Human Sign-off — All agent recommendations require human approval before any system action is taken

Analyst Productivity +40% — Reporting cycle time reduced 40–50%; analysts shift to higher-value interpretation work

Training Underway — Ongoing AI adoption training embedded in finance team onboarding and development plans

ERP Connected — Read-only ERP API connection established; agent pulls GL, AR, AP data in real time
BI Layer Live — Data warehouse or BI platform (Power BI / Looker) feeding agent models with clean data
Master Data Cleaned — Critical vendor and customer master data deduplication completed; MDM governance in place
Single Source of Truth — Agreed financial data definitions and chart of accounts adopted across all systems

Intelligent Automation
Agents handle end-to-end transactional finance processes with humans reviewing exceptions only — straight-through processing active
Automation
Avg Maturity Indicator 50%

Continuous Control Testing — SOX controls tested on every transaction sample; control results logged automatically in audit tool
Fraud Detection Live — ML-based Fraud & Duplicate Detection Agent screens all AP payments before release; near-zero duplicate rate
Segregation of Duties — Agent action boundaries enforce SoD rules programmatically; no single agent can both approve and post
Exception Escalation — All out-of-policy transactions auto-escalated to appropriate approver with AI reasoning attached

Agent CoE Forming — Finance Centre of Excellence (CoE) established to govern agent performance, exceptions, and expansion

ERP Write-back Enabled — Agents post journal entries, update records, and trigger payments directly in ERP via authenticated APIs
Multi-system Integration — O2C, P2P, and R2R agents share customer, vendor, and GL data via integration middleware layer
Real-time Event Triggers — Agent workflows triggered by ERP events (invoice received, payment cleared) rather than scheduled batches
Data Quality Agents Live — Customer and Vendor Data Quality Agents run continuously; master data accuracy exceeds 95%
